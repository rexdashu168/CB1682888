## 🚀 部署步驟

### GitHub Pages 部署

```bash
# 1. 創建 GitHub repository

# 2. 上傳以下檔案到 repository 根目錄：
- index.html  (必要)
- data.json   (必要)
- README.md   (建議)

# 3. 啟用 GitHub Pages
Settings → Pages → Source: main branch → Save

# 4. 等待 2-3 分鐘後訪問
https://你的帳號.github.io/repo名稱/
```

### 本地測試

```bash
# 方法1：使用 Python
python -m http.server 8000
# 然後開啟 http://localhost:8000

# 方法2：使用 Node.js
npx http-server
# 然後開啟顯示的網址
```

⚠️ **注意：不能直接雙擊 index.html 開啟（會有 CORS 錯誤），必須透過 HTTP 伺服器！**

## ⚠️ 常見問題

### Q1: 為什麼打開後沒有資料？
**A:** 請確認：
1. index.html 和 data.json 在同一目錄
2. 透過 HTTP 伺服器開啟（不是直接雙擊）
3. 等待資料載入（約 10-20 秒）

### Q2: 出現 "無法載入資料檔案" 錯誤？
**A:** 檢查：
1. data.json 檔案是否存在
2. 檔案名稱是否正確（注意大小寫）
3. 瀏覽器 Console 查看詳細錯誤

### Q3: GitHub Pages 部署後顯示空白？
**A:** 等待幾分鐘，GitHub Pages 需要時間建置
