# 🎯 台股AI智能查詢系統

## 📋 專案說明

**2269家上市櫃 | 66篇深度報告 | 1021家CB完整資訊 | 297家CB競拍完整資訊**

一個整合全台上市櫃公司的完整資訊查詢系統，提供：
- **2,269 家公司** 完整基本資料
- **66 篇深度報告** 專業研究分析
- **1,021 家公司** 可轉債完整資訊
- **297 家公司** CB競拍完整數據

## 🎯 系統功能

### 🔍 智能查詢
- 股票代號查詢（如：2330）
- 公司名稱查詢（如：台積電）
- 即時搜尋，秒速顯示

### 📋 六大查詢維度

1. **📋 公司總覽**
   - 公司基本資料
   - 產業地位說明
   - 股價交易資訊
   - 財務快覽
   - 產品結構
   - 成長動能

2. **💰 財務表現**
   - 3Q25完整財務數據
   - YoY/QoQ變化分析
   - 4Q25指引

3. **📦 產品結構**
   - 主要產品線
   - 產品表現分析
   - 成長驅動因素
   - 2026展望

4. **📊 研究報告**
   - 66篇深度報告
   - 營運重點分析
   - 策略方向解析
   - 重要QA摘要
   - Rex觀察評論

5. **🎯 可轉債資訊**
   - 目前掛牌中（344檔）
   - 競拍數據（297檔）
   - 歷史紀錄（2,232檔）
   - 轉換價、得標價分析

6. **⏰ 關鍵時程**
   - 時間軸追蹤
   - 重要事件
   - 影響評估

## 🎯 可轉債功能

### 📊 完整CB資訊

#### 🔥 目前掛牌中（344檔）
- 轉換價格、發行額度
- 剩餘餘額、到期日
- 年期、票面利率

#### 📊 競拍數據（297檔）
- 最低/平均得標價
- 投標倍數
- 承銷券商
- 發行規模

#### 📜 歷史紀錄（2,232檔）
- 掛牌最高/最低
- 資金用途
- 完整發行紀錄

### 💡 CB查詢範例

**查詢有CB的公司：**
1. 搜尋公司（如：2330 台積電）
2. 點擊「🎯 可轉債」頁籤
3. 查看完整CB資訊
